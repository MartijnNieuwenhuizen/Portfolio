GO

Front-endJavaSriptSASS

HvA 2016

Go is a platform that provides possible destinations for Digital Nomads.

You can limit the possible destinations with the filters (Wifi, Cost of Living, Climate). After filtering the city’s it’s possible to compare them, save them and make a choice about your next destination. All the choices made in this project are based research by the entire team.

The Team

  • Front-end: Dylan Vens & Martijn Nieuwenhuizen
  • Design: Heleen Snoeck & Erik Odijk & Maaike Tol
  • UX: Erik Odijk
  • Animations: Maaike Tol

For the Nerds

The Go application has a very modular approach. All the HTML, CSS and JavaScript has a modular setup with the intention that it's usable in every upcoming project without altering the base code. So if you want to change the appearance, you will only have to change one file.

We also used BAM (which I will never do again) and a nice gulp setup. You can find everything on GitHub.
